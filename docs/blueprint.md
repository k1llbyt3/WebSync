# **App Name**: FocusFlow

## Core Features:

- AI-Powered Task Prioritization: Automatically prioritize tasks extracted from various inputs using AI analysis and reasoning to provide scores based on deadlines, effort, and trends. The tool will dynamically adjust priorities.
- Meeting Transcription & Analysis: Ingest meeting transcripts and files to extract key information such as action items, dates, and summaries, and suggest tasks
- Code Generation: Generate code snippets, tests, and documentation based on task descriptions using AI for various languages. AI can suggest code based on previous behavior.
- Secure Authentication & Onboarding: Implement secure user authentication with JWT, including login, signup, and profile management with preference settings (theme, notifications, default priority).
- Kanban Task Management: Implement a Kanban board for task management with drag-and-drop functionality, sorting, filtering, and search by priority, status, tags, and date. Persist user information in an SQLite database
- Real-time Collaboration Hubs: Enable real-time collaboration through shared Kanban boards and invite systems to streamline team workflows.
- Pulse Dashboard with AI Insights: Provide an at-a-glance dashboard with metrics on tasks, trends, and AI-driven insights for efficiency improvements.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) for a calming and focused feel.
- Accent color: Seafoam Green (#98FF98) for gentle, eye-catching highlights.
- Background color: Light Gray (#F5F5F5) for a comfortable and unobtrusive base.
- Body and headline font: 'Open Sans' for a clean, readable, and friendly look.
- Use brightly colored and slightly larger 'Fluent UI' icons throughout the interface to draw the eye.
- Employ a clean and spacious design. Cards should have rounded corners and subtle shadows.
- Subtle scaling and fade-in animations on buttons and card backgrounds to provide visual feedback and delight. Use smooth transitions.
